Importación desde cualquier asistente IA

AnGoTest permite una integración flexible con cualquier asistente de IA externo (ChatGPT, Claude, Gemini, etc.). El administrador puede solicitar al asistente que genere un test completo siguiendo la estructura JSON definida, y luego importarlo directamente en el panel de administración con un solo clic.

"Genera un test de 10 preguntas sobre el sistema solar para nivel principiante con 4 opciones cada una, en formato JSON compatible con AnGoTest"

Estructura JSON para importación

Cualquier asistente de IA puede generar tests para AnGoTest siguiendo esta estructura exacta. El sistema valida automáticamente el formato y crea el test con todas sus preguntas y respuestas. 

{{ jsonStructure | json }}

Reglas de validación

  • Una única respuesta correcta por pregunta
  • Mínimo 2 respuestas por pregunta (recomendado 3-4)
  • Título y descripción obligatorios
  • La jerarquía de temas puede ser existente o nueva

Campos opcionales

  • Si no se incluye jerarquía, se usará "General"
  • El nivel puede ser "Principiante", "Intermedio" o "Avanzado"
  • La descripción puede omitirse

Ejemplo de prompt para cualquier asistente IA

"Genera un test educativo en formato JSON compatible con la plataforma AnGoTest con las siguientes características:

- Tema principal: Historia

- Subtema: Edad Media

- Tema específico: La Reconquista española

- Nivel: Intermedio

- 8 preguntas

- 4 opciones por pregunta

- Una única respuesta correcta por pregunta

El JSON debe incluir title, description, main_topic, sub_topic, specific_topic y un array de questions con question_text y answers (con answer_text e is_correct)."
